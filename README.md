### GitHub Pages

1. Go to your repository Settings
2. Navigate to Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/ (root)` folder
5. Click Save

The site will be available at: `https://valueaddersworld.github.io/Maxwell-Cecil-Lagos-operations-strategy/`

### Local Development

Simply open `index.html` in a web browser. No build process required.

```bash
# Clone the repository
git clone https://github.com/ValueaddersWorld/Maxwell-Cecil-Lagos-operations-strategy.git

# Navigate to directory
cd Maxwell-Cecil-Lagos-operations-strategy

# Open in browser (Windows)
start index.html

# Or serve with a local server
npx serve .
```
